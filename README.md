# Описание проекта

[requirements.txt](https://github.com/MargaritaArunova/decepticon/blob/main/requirements.txt) - python packages
 
[model.py](https://github.com/MargaritaArunova/decepticon/blob/main/model.py) - модель, построеная на основе архитектуры из статьи "Attention is All You Need"

[train.py](https://github.com/MargaritaArunova/decepticon/blob/main/train.py) - обучение и тестирование модели

[dataset.py](https://github.com/MargaritaArunova/decepticon/blob/main/dataset.py) - модели для обработки датасетов

[main.py](https://github.com/MargaritaArunova/decepticon/blob/main/main.py) - запуск обучения модели и получение скора

[metric_utils.py](https://github.com/MargaritaArunova/decepticon/blob/main/metrics_utils.py) - вспомогательные функции для работы с загрузкой текстов и обученных моделей

[experiments](https://github.com/MargaritaArunova/decepticon/blob/main/experiments) - эксперименты, процесс обучения с использованием разных токенизаторов

[datasets](https://github.com/MargaritaArunova/decepticon/blob/main/datasets) - модели датасетов

[prod](https://github.com/MargaritaArunova/decepticon/blob/main/prod) - результаты работы обученной модели и сама модель
